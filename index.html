Firebase Einrichtung

Für die Synchronisation zwischen euren Geräten tragt hier eure Firebase-Konfiguration ein. Ohne Firebase werden alle Daten nur lokal gespeichert.

Gehe auf console.firebase.google.com und erstelle ein kostenloses Konto
Klicke auf „Projekt erstellen" → gib einen Namen ein (z.B. „Familienküche")
Klicke auf das Web-Icon </> → App registrieren → du bekommst eine Konfiguration
Gehe zu „Firestore Database" → Datenbank erstellen → Testmodus wählen
Trage die Werte unten ein und speichere
